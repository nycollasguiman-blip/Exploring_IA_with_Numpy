# Overview

This repository is a collection of implementations of Artificial Intelligence and Machine Learning algorithms developed using only NumPy and standard Python libraries.

The purpose of this project is not to create production-ready models, but to gain a deep understanding of the mathematics and logic behind modern AI. Every algorithm is implemented from scratch to explore how it works internally before using high-level frameworks.

While my other AI repositories focus on practical applications using libraries such as TensorFlow, PyTorch, and Scikit-learn, this repository is dedicated exclusively to studying the foundations that those libraries abstract away.

# Objectives
Understand the mathematics behind AI algorithms.
Implement models from scratch using NumPy.
Learn optimization and gradient-based learning.
Explore neural network fundamentals.
Build a strong theoretical foundation for future AI projects.
